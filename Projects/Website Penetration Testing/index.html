Website Penetration Testing

In this academic project, the goal is to perform penetration testing on the website of company xxxxx, which is a partner of Politeknik Negeri Batam. In this project, we were asked to test two partner websites of Polibatam to identify security vulnerabilities with a domain-based scope. After finding several vulnerabilities, we prepared a report and recommendations for improvements.

Some of the vulnerabilities we discovered include:

  • SQL Injection
  • Session Hijacking
  • Broken Access Control
  • IDOR
