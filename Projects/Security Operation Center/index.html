SIEM

Pada peoject ini kami membuat sebuah architecture monitoring menggunakan vmware yang bertujuan untuk memantau aktivitas serangan dan melakukan automisasi analisa.

Program yang kami gunakan pada project ini :

  • Wazuh
  • TheHive
  • Cortex
  • Shuffle

Wazuh memiliki sebuah agent yang akan mengirimkan semua log server kepada wazuh server menjadi sebuah alert lalu dengan automisasi shuffle alert tersebut dikirimkan kepada thehive untuk membuat ticketing kepada cortex Cortex akan melakukan analisa apakah alert itu berbahaya atau tidak dengan cara mencocokan ip atau file yang dikirimkan oleh penyerang tersebut terdaftar dalam AbuseIP dan Virustotal.
